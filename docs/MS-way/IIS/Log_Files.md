### Places
```bash
# First one: Default place to keep Site's logs (But it can be changed in IIS Manager)
%SystemDrive%\inetpub\logs\LogFiles

# HTTP Error logs:
%SystemDrive%\Windows\System32\LogFiles\HTTPERR

# If you cann't find log you should be sure that they are enabled:
# * Go to "Programs and Features"
# * Click on "Turn Windows features on or off"(on the left side)
# * Select "Internet Information Services\World Wide Web Services\Health and Diagnostics\HTTP Logging"
```
