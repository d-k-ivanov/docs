Rewind tape drive

mt -f /dev/st0 rewind

Backup directory

tar -czvf /dev/st0 /<directory_for_backup>

List files on tape

tar -tzf /dev/st0

Restore

mt -f /dev/st0 rewind
tar -xzf /dev/st0 <file_or_dir_to_restore>

Unload

mt -f /dev/st0 offline

Status

mt -f /dev/st0 status

Erase

mt -f /dev/st0 erase

Get position

mt -f /dev/st0 tell

Moves

# End
mt -f /dev/nst0 eod
# Previous
mt -f /dev/nst0 bsfm 1
# Next
mt -f /dev/nst0 fsf 1
